Reports Vue components whose <script setup> block is too long or defines too many functions.

Oversized components are a sign of mixed responsibilities. Extracting logic into composables makes the component easier to understand and test.

Default thresholds (based on ReactSniffer 95th-percentile data):

  • Script block lines: 128
  • Functions: 4

Vue reactivity primitives (computed, watch, watchEffect) are not counted as functions.
